# Scoring Model

The bot evaluates sea conditions using a custom score from 0 to 100.

## Parameters

- Wave height
- Wind speed
- Wind gusts
- Rain probability
- Forecast consistency
- Time window quality

## Logic

The score rewards:

- low wave height;
- weak and stable wind;
- limited gusts;
- low rain probability;
- consistent forecast data.

The goal is not to provide a professional marine forecast, but to support a practical decision for SUP activities.
